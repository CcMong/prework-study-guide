HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Heading tags go from h1 to h6. Only one h1 tag allowed.
  • The p element is used to wrap a paragraph or block of text.
  • The img element can contain many attributes, but most important is src.
  • The src attribute indicates the location of the image file.
  • Semantic elements indicate the role of their content, and provide content structure.

CSS

  • Cascading Style Sheets (CSS) give the webpage content style.
  • CSS describes how HTML elements are to be displayed on screen.
  • Three ways to style a webpage are Inline CSS, Internal Style Sheet, and External Style Sheet.
  • External CSS makes it possible to alter the look of a webpage by changing just one file.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can style elements by using the elements as selectors, or using the class attribute as a selector.

Git

  • Git is an open source version control system.
  • Git allows multiple developers to work asynchronously on a project.
  • git status: Checks which branch we are currently on.
  • git checkout -b branch-name: Creates a new branch and switches to it.

JavaScript

  • JavaScript is the programming language of the web.
  • JavaScript allows developers to program the behaviour of webpages.
  • It is responsible for making webpages more responsive and interactive for users.
